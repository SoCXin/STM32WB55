## [描述](https://github.com/mcuyun/CA53/wiki) 

基于ARMv8-A 64架构，单个核心的功耗不超过0.13W，最小的64位处理器，能够支持32位的ARMv7代码和64位代码的AArch64执行状态，A53相比A9最大的优势在于程序执行效率，是当前国产芯片的主流架构之一。
